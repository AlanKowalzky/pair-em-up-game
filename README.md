# Pair 'em Up Game Repository

This repository contains the Pair 'em Up puzzle game project.

## Branches

- `main` - Empty main branch (as per project requirements)
- `dev` - Development branch containing the complete game implementation

## Development

Switch to the `dev` branch to see the complete implementation:

```bash
git checkout dev
```
